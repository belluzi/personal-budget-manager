# Personal Budget Manager 💰

![GitHub last commit](https://img.shields.io/github/last-commit/belluzi/app-orcamento-pessoal?style=for-the-badge)
![GitHub repo size](https://img.shields.io/github/repo-size/belluzi/app-orcamento-pessoal?style=for-the-badge)

Aplicação web para gerenciamento de despesas pessoais, permitindo cadastro, consulta, filtragem e exclusão de registros financeiros de forma simples e intuitiva.

O projeto foi desenvolvido com foco em manipulação de dados no frontend, organização de lógica com JavaScript e persistência local utilizando `localStorage`.

## Funcionalidades 🚀

✔️ **Cadastro de despesas**  
Registro de despesas contendo data, categoria, descrição e valor.

✔️ **Consulta de registros**  
Listagem dinâmica das despesas cadastradas em tabela organizada.

✔️ **Filtros avançados**  
Pesquisa por ano, mês, dia, categoria, descrição ou valor.

✔️ **Remoção de despesas**  
Exclusão de registros diretamente pela interface.

✔️ **Persistência local de dados**  
Armazenamento dos dados utilizando `localStorage`, mantendo as informações salvas no navegador.
